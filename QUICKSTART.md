# LinguaCoach - Quick Start Guide

## 🚀 Getting Started

### Running the App
```bash
cd linguacoach
npm run dev
```
App opens at: `http://localhost:5174/`

---

## 📖 Complete User Flow

### Step 1: Start a Micro-Mock (2-3 minutes)
1. Go to **Micro-Mock** tab
2. Select test type (IELTS or Duolingo)
3. Set your goal score (e.g., 7.0)
4. Click "Start Writing" or "Start Speaking"

### Step 2: Writing Practice
1. Read the prompt
2. Write your response (150-250 words recommended)
3. Click "Generate Coach Feedback"
4. **You'll see:**
   - 📊 Band score (0-9)
   - 📈 Rubric breakdown (4 criteria)
   - ✅ Your strengths
   - ⚠️ Top fixes needed
   - 📝 Highlighted sentences with feedback
   - ✏️ Example rewrite
   - 🎯 6 personalized drills
   - 💡 Next best action

### Step 3: Speaking Practice
1. Read the speaking prompt (45-60 seconds)
2. **Option A:** Click "Start Live Transcript" and speak (Chrome recommended)
3. **Option B:** Paste your transcript manually
4. Click "Analyze Speaking"
5. **You'll see:**
   - 📊 Band score
   - 📈 Rubric: Fluency, Coherence, Vocabulary, Pronunciation
   - 📊 Metrics: WPM, fillers/min, word count
   - ⚠️ Issues identified
   - 💪 Speaking drills
   - 💡 Next best action

### Step 4: Delivery Practice (Video Mode)
1. Go to **Delivery** tab
2. Click "📹 Start Camera" (allow browser camera access)
3. Practice your answer with good eye contact
4. Click "⏹️ Stop & Analyze"
5. **You'll see:**
   - 👁️ Eye contact score (0-100)
   - 🎯 Movement control score
   - 💡 Coach notes on your delivery

### Step 5: Generate Study Plan
1. Go to **Learning** tab
2. Click "📚 Generate Study Plan"
3. System creates lessons based on your top issues
4. Each lesson includes:
   - 📝 3 study tips
   - 📌 Real examples from your work
   - ✅ Completion tracking

### Step 6: Track Progress
1. Go to **Dashboard** tab
2. See your progress snapshot:
   - Total attempts
   - Average writing band
   - Average speaking band
3. Review your memory:
   - Top grammar issues
   - Top speaking habits
   - All coach notes in order

## 🐛 Troubleshooting

### Speech Recognition Not Working
- ✅ Use Chrome browser (best support)
- ✅ Check browser microphone permissions
- ✅ Paste transcript manually instead

### Camera Not Showing
- ✅ Allow camera access when prompted
- ✅ Check browser permissions (Settings → Privacy → Camera)
- ✅ Try different browser (Chrome, Firefox, Edge)

### Data Not Saving
- ✅ Check if LocalStorage is enabled
- ✅ Try Private/Incognito mode (stores in session)
- ✅ Clear cache and reload

### Scores Seem Off
- ✅ Scores are heuristic-based (not real IELTS)
- ✅ For real scoring, use official prep materials
- ✅ Coaching insights are directionally accurate

## 📚 Learning Tips

### Writing Practice
1. **Before:** Read prompt 2x, plan structure
2. **During:** Use 4-paragraph template (Intro → 2 Bodies → Conclusion)
3. **After:** Review drills and rewrite example

### Speaking Practice  
1. **Before:** Plan 4 points (Opinion, Reason, Example, Wrap-up)
2. **During:** Record and listen back
3. **After:** Practice targeted drills from feedback

### Delivery Practice
1. **Before:** Practice with confidence
2. **During:** Keep eye contact, control movement
3. **After:** Note improvements from feedback

### Lesson Study
1. Read goal and tips
2. Study your real examples
3. Do the practice
4. Mark complete when confident

## 🎓 IELTS Tips

**Writing (Task Response)**
- 180-250 words minimum
- 4 clear paragraphs
- Directly answer the question
- Use specific examples

**Writing (Coherence)**
- Use linking words (however, therefore, for example)
- Topic sentences for each paragraph
- Logical flow between ideas

**Writing (Vocabulary)**
- Avoid repetition
- Use precise words
- Include collocations (natural phrases)

**Writing (Grammar)**
- Check article usage (a/an/the)
- Maintain consistent tense
- Correct sentence structures
- Subject-verb agreement

**Speaking (Fluency)**
- 90-165 words per minute
- Minimize fillers (um, uh, like)
- Use linking phrases to connect ideas
- Pause naturally, don't rush

**Speaking (Coherence)**
- Structure: Opinion → Reason → Example → Wrap-up
- Use discourse markers (first, second, however, overall)
- Develop ideas with examples
- Conclude clearly
